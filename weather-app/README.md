A simple weather application built with HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Uses OpenWeatherMap API
- Displays temperature, humidity, wind speed, and description

## Technologies
- HTML
- CSS
- JavaScript
- Fetch API

## How to use
1. Enter a city name
2. Click "Get Weather"
3. View current weather data

## Setup

1. Create a free account at https://openweathermap.org
2. Get your API key
3. Replace `YOUR_API_KEY_HERE` in `script.js`
